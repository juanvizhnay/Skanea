# Ejemplos de Exportación en el Chat de Skanea

¡Ahora puedes pedirle a Skanea que genere archivos directamente desde el chat! Aquí tienes algunos ejemplos de cómo usar esta nueva funcionalidad:

## 🔥 Ejemplos de Comandos

### 📄 PDFs
```
"Genérame un resumen de Blancanieves en PDF"
"Crea un reporte PDF sobre inteligencia artificial"
"Haz un documento PDF con las ventajas del desarrollo web"
```

### 📝 Documentos Word (DOCX)
```
"Crea un documento Word sobre los beneficios del ejercicio"
"Genérame un reporte DOCX del proyecto"
"Haz un archivo Word con una lista de tareas"
```

### 📊 Hojas de Excel (XLSX)
```
"Crea una hoja Excel con datos de ventas"
"Genérame un archivo XLSX con presupuesto mensual"
"Haz una tabla Excel de empleados"
```

### 📋 CSV
```
"Crea un CSV con lista de productos"
"Genérame datos CSV de clientes"
"Haz una tabla CSV con inventario"
```

### 📄 Texto (TXT)
```
"Genérame un archivo de texto con tips de programación"
"Crea un TXT con notas de la reunión"
"Haz un archivo texto con el código de conducta"
```

## 🎯 Palabras Clave que Detecta

La IA detecta automáticamente cuando quieres generar un archivo usando estas palabras:

- **Acciones**: genera, generar, crea, crear, haz, hacer, dame, exporta, exportar
- **Formatos**: PDF, DOCX, Word, TXT, texto, CSV, Excel, XLSX
- **Contexto**: archivo, documento, reporte, informe, resumen, lista, tabla

## 🚀 ¿Cómo Funciona?

1. **Escribes tu solicitud** en el chat normal de Skanea
2. **La IA detecta** automáticamente que quieres un archivo
3. **Genera el contenido** según tu solicitud
4. **Crea el archivo** en el formato solicitado
5. **Te muestra el contenido** + un botón de descarga

## 💡 Tips para Mejores Resultados

### ✅ Buenos Ejemplos:
- "Crea un PDF con un resumen de las mejores prácticas de JavaScript"
- "Genérame un Excel con datos de ejemplo para un e-commerce"
- "Haz un documento Word sobre metodologías ágiles"

### ❌ Evita:
- Solicitudes muy vagas: "Dame algo en PDF"
- Sin especificar formato: "Crea un resumen" (no especifica PDF/Word/etc.)

## 🎨 Lo Que Verás en el Chat

Cuando pidas un archivo, verás:
1. **Respuesta normal** con el contenido
2. **Tarjeta de archivo** con:
   - 📎 Icono de archivo
   - Nombre del archivo
   - Tipo y formato
   - Botón de descarga azul

## 📁 Ubicación de Archivos

Los archivos se guardan en:
- Backend: `backend/exports/`
- Se limpian automáticamente después de 24 horas

¡Prueba ahora mismo escribiendo algo como "Crea un PDF con un resumen de Blancanieves"! 🎉
